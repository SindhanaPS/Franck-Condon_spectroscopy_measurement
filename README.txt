Code accompanying manuscript "Precision Franck-Condon spectroscopy from highly-excited 
vibrational states" (https://doi.org/10.48550/arXiv.2401.10384)

The code 'single_mode.py' with:

1) flag1 = 0 generates data for Fig. 1d-f
1) flag1 = 1 with flag2 = 0,1,2 generates data for eta = 0.1,0.025,0.011, respectively, in Fig. 2
2) flag1 = 2 generates data for Fig. S1

The code 'multi_mode.py' generates data for Figures 3a-e

The code 'Morse_single_mode.py' generates data for Fig. S2-3
1) flag1 = 0 uses CaH parameters and generates data for Fig. S2a and S3a
2) flag1 = 1 uses SrF parameters and generates data for Fig. S2b and S3b

The code 'plot_DHO.py' with:

1) flag1 = 0 generates Figure 1a
2) flag1 = 1 generates Figure 1b
3) flag1 = 2 generates Figure 1c

The code 'plot_single_mode.py' with:

1) flag1 = 0 plots Figures 1d-f
2) flag1 = 1 with flag2 = 0,1,2 processes data and generates 'DdN_eta_1.txt', 'DdN_eta_4.txt' and 'DdN_eta_9.txt', respectively, for eta = 0.1,0.025,0.011 required for Figure 2
3) flag1 = 2 plots Figure 2
4) flag1 = 3 plots Figure S1
5) flag1 = 4 plots Figure S2a-b
6) flag1 = 5 plots Figure S3a-b

The code 'plot_multi_mode.py' plots Figures 3a-e
